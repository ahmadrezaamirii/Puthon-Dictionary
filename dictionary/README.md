# Dictionary

***
### First, to use a dictionary, we need to use curly braces{}.

````
fruit_price = 
{
    apple:20
    banana:25
    orange:30
}
````
### Then we can use .format on them.
#### for change the price:
````
fruit_price.replace(banana) = 28
````
#### for remove the somthing
````
fruit_price.remove(apple)
````
